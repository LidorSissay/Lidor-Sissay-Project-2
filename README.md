# Cryptocurrency Dashboard

A modern cryptocurrency dashboard built with React + TypeScript.

The application allows users to:
- Browse cryptocurrencies
- Select favorite coins
- View live reports and charts
- Get AI-powered investment recommendations

---

## Technologies

- React
- TypeScript
- Redux Toolkit
- React Router
- Axios
- Chart.js
- Vite

---

## Features

### Home Page
- Responsive cryptocurrency grid
- Coin selection using toggle controls
- More Info modal for detailed coin information

### Live Reports
- Real-time cryptocurrency chart updates
- Multiple selected coins support

### AI Recommendations
- AI-generated cryptocurrency investment recommendations
- NVIDIA API integration
